2 Conv(filter, in => out, σ = identity; stride = 1, pad = 0, dilation = 1, groups = 1, [bias, initW])
3
4 Standard convolutional layer.
5
6 # Arguments
7
8 * `filter` is a tuple of integers specifying the size of the convolutional kernel
9 * `in` and `out` specify the number of input and output channels.
10
11 Image data should be stored in WHCN order (width, height, channels, batch). In other words, a 100×100 RGB image would be a `100×100×3×1` array, and a batch of 50 would be a `100×100×3×50` array. This has `N = 2` spatial dimensions, and needs a kernel size like `(5,5)`, a 2-tuple of integers. To take convolutions along `N` feature dimensions, this layer expects as input an array with `ndims(x) == N+2`, where `size(x, N+1) == in` is the number of input channels, and `size(x, ndims(x))` is (as always) the number of observations in a batch.
12 * `filter` should be a tuple of `N` integers.
13 * Keywords `stride` and `dilation` should each be either single integer, or a tuple with `N` integers.
14 * Keyword `pad` specifies the number of elements added to the borders of the data array. It can be
15 - a single integer for equal padding all around,
16 - a tuple of `N` integers, to apply the same padding at begin/end of each spatial dimension,
17 - a tuple of `2*N` integers, for asymmetric padding, or
18 - the singleton `SamePad()`, to calculate padding such that `size(output,d) == size(x,d) / stride` (possibly rounded) for each spatial dimension.
19 * Keyword `groups` is expected to be an `Int`. It specifies the number of groups to divide a convolution into.
20
21 Keywords to control initialization of the layer:
22 * `initW` - Function used to generate initial weights. Defaults to `glorot_uniform`.
23 * `bias` - The initial bias vector is all zero by default. Trainable bias can be disabled entirely by setting this to `false`.
